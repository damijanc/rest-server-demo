# simple-rest
Example rest service using Symfony 3, Doctrine and MariaDB



## Installation:

* Run in project root ```composer install```
* Setup database ```mysql -u root < ./Model/generation.sql```


## Tests

To run tests run ```./vendor/bin/phpunit``` in project root
